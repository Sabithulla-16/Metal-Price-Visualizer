# Metal Price Visualizer

A web-based application developed to visualize and analyze metal price data using interactive charts. The project focuses on presenting numerical data in a clear and structured format to help observe trends and variations effectively.

## Overview
This project demonstrates the use of frontend web technologies to build a simple yet effective data visualization interface. It emphasizes practical implementation, logical structuring of data, and user-friendly presentation.

The application allows users to view metal price trends graphically and understand price variations over time through visual representation.

## Key Features
- Interactive graphical visualization of metal price data  
- Clear and structured data presentation  
- Simple and intuitive user interface  
- Export functionality for generated reports (PDF)
